## ServedJS

This is ServiceJS extension library to add some basic services for systems.

### Why?

It is tools collections into services.

### API

Services:

    $log

    $browser

    $localStorage

    $sessionStorage

    $strings

### Setup

npm install && bower install

### Run server for examples

npm run start

or

npm start

or

mvn jetty:run

http://localhost:3000/

### Unit testing

npm run unit

### Run example program

npm run program
